API Key Tracker

A Node.js Web Application

For more applications, please check my Applications Page.

API Key Tracker automatically encrypts and organizes all your vendor API keys into a secure, searchable dashboard. By linking vendors to accounts and accounts to individual keys, you can manage your entire API infrastructure from a single, self-hosted application with AES-256-CBC encryption at rest.

Features

  • Targeted Organization: Vendors → Accounts → Keys — a clean, normalized hierarchy that scales.
  • AES-256 Encryption: Every API key is encrypted at rest. Each keys file gets its own unique encryption key.
  • Flexible Key Files: Create, switch between, and manage multiple key file datasets — perfect for different projects or environments.
  • Instant Search & Filter: Full-text search across accounts, projects, names, purposes, and notes. Dropdown filters for vendor, account, status, and project.
  • Intuitive Interface: Clean, tabbed layout with DataTables-powered sorting, pagination, and one-click copy.
  • Efficient: Lightweight Node.js + Express server with file-based storage — no database required.
  • Docker Ready: Includes Dockerfile and docker-compose.yml for one-command deployment.
  • Dark/Light Theme: Toggle between dark and light mode with preference saved automatically.
  • Zip Export: Export decrypted data and encryption key as a zip file for backup or migration.

Installation

  1. Clone: Clone the repository from GitHub: 
    git clone https://github.com/jorper98/api-key-tracker.git
cd api-key-tracker
  2. Install: Run npm install to install dependencies.
  3. Start: Run node server.js and open http://localhost:3000 in your browser.

Or with Docker:

docker-compose up -d

Use Cases

  • Developer Tooling: Keep all your SaaS API keys organized and encrypted in one place.
  • Multi-Project Management: Separate key files per project or environment (dev, staging, production).
  • Team Collaboration: Share a centralized key registry instead of sending keys over Slack or email.
  • Security Compliance: Ensure keys are encrypted at rest and never stored in plain text or version control.
  • Freelancer Dashboard: Manage client API keys across multiple vendors and accounts with full traceability.
  • DevOps & Infrastructure: Track which keys belong to which accounts and vendors for audit purposes.
  • Onboarding/Offboarding: Quickly see which keys are active, inactive, or need rotation when team members change.
  • Backup & Migration: Export your entire key database with encryption keys for safe backup or transfer.

Usage

  • Open the application in your browser at http://localhost:3000.
  • Vendors tab: Add and manage your SaaS vendor relationships.
  • Accounts tab: Create vendor accounts with email, auth type, billing status, and notes.
  • Keys tab: Add API keys linked to accounts with status, project, name, purpose, and encrypted storage.
  • Use the search bar and dropdown filters to quickly find any key or account.
  • Click the eye icon to reveal a key, or the copy icon to copy it to your clipboard.
  • Create multiple key files from the header dropdown to isolate different projects.
